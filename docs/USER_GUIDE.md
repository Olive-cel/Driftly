## Inscription

### Étape 1: Aller sur la page d'inscription

```
URL: https://driftly-two.vercel.app/signup
```

### Étape 2: Remplir le formulaire

```
Email: votre@email.com
Mot de passe: (Au moins 8 caractères)
Confirm: Re-saisir le mot de passe
```

### Étape 3: Cliquer "S'inscrire"

- Email de confirmation envoyé
- Vérifiez votre boîte mail
- Cliquez le lien de confirmation

### Étape 4: Compléter le profil

```
Nom: Votre Nom
Prénom: Votre Prénom
Style de voyage: (Budget / Modéré / Premium)
Budget préféré: €€€ (ex: 2000€)
Intérêts: (Culture, Nature, Plages, Cuisine, etc.)
```

**Profil créé!**

## Créer un Voyage

### Méthode 1: Via le tableau de bord

```
1. Bouton "+ Nouveau Voyage"
2. Formulaire:
   - Destination: Paris
   - Ville de départ: Lyon
   - Date de départ: 2026-08-15
   - Date de retour: 2026-08-22
   - Budget: 2500€
   - Voyageurs: 2
   - Style: Culturel
   - Intérêts: Musées, Cuisine
3. Cliquez "Créer le Voyage"
```

### Méthode 2: Via les Inspirations

```
1. Allez sur Inspirations
2. Sélectionnez une destination
3. Bouton "Créer un Voyage"
4. Formulaire pré-rempli avec destination + style
5. Ajustez si besoin
6. Validez
```

**Voyage créé!**

## Générer un Itinéraire IA

### Sur la page détail du voyage

```
1. Onglet "Itinéraire"
2. Bouton "Générer avec l'IA"
3. Attendre 10-30 secondes
4. L'IA génère:
   - Jours détaillés (Jour 1, 2, 3...)
   - Activités recommandées
   - Heures suggérées
   - Durée de trajets
   - Budget par jour
5. Parcourez les suggestions
```

### Structure de l'itinéraire

```
Jour 1: Arrivée & Découverte
├─ 14:00 - Arrivée à l'aéroport
├─ 15:00 - Check-in hôtel (45 min trajet)
├─ 16:30 - Repos et détente
├─ 18:30 - Dîner restaurant typique
└─ Budget du jour: 150€

Jour 2: Visites Principales
├─ 09:00 - Visite Louvre (3h)
├─ 12:30 - Déjeuner Bistro (1h30)
├─ 14:00 - Musée Orsay (2h)
├─ 17:00 - Balade Seine
└─ Budget du jour: 200€
```

## Chercher des Vols

### Page Détail Voyage > Onglet "Vols Recommandés"

```
1. Liste de vols trouvés
2. Chaque vol affiche:
   - Compagnie aérienne
   - Horaires (Départ > Arrivée)
   - Durée + escales
   - Prix par personne
3. "Meilleur prix" badge sur le moins cher
4. Bouton "Réserver" (lien externe)
```

### Exemple

```
Air France
08:00 (CDG) → 10:00 (FCO)
2h direct
150€ par personne
Meilleur prix
```

## Trouver des Hôtels

### Page Détail Voyage > Onglet "Hôtels Recommandés"

```
1. 3 options affichées:
   - Économique (~60€/nuit)
   - Meilleur choix (~120€/nuit)
   - Premium (~250€/nuit)
2. Chaque hôtel:
   - Image réelle (via Pexels)
   - Note (3.5-5.0)
   - Amenities (WiFi, Petit-déj, etc.)
   - Prix/nuit et prix total
   - Description brève
3. Bouton "Sélectionner"
```

## Explorez les Inspirations

**URL**: [https://driftly-two.vercel.app/dashboard/inspiration](https://driftly-two.vercel.app[/](https://driftly.app/dashboard))

### Découvrir des Destinations

```
1. Inspirations basées sur votre profil
2. Chaque carte affiche:
   - Image magnifique (Pexels)
   - Destination + Pays
   - Note
   - Description courte
   - Budget estimé
   - Intérêts
3. Cliquez image → Voir détails
4. Bouton ❤️ → Ajouter aux favoris
5. Bouton "Créer Trip" → Pré-remplit un voyage
```

### Filtrer

- Par style: Budget, Modéré, Premium
- Par intérêts: Culture, Plages, Nature, Cuisine
- Par saison

## Gérer les Favoris

**URL**: [https://driftly-two.vercel.app/dashboard/favorites](https://driftly-two.vercel.app[/](https://driftly.app/dashboard))

### Ajouter un Favori

```
1. Inspiration ou Trip
2. Cliquez ❤️ 
3. Cœur devient rouge
4. Item ajouté à "Mes Favoris"
```

### Retirer un Favori

```
1. Inspirations page ou Favoris page
2. Cliquez ❤️ rouge
3. Cœur devient gris
4. Item retiré
```

### Consulter les Favoris

```
1. Onglet "Mes Favoris"
2. Liste de:
   - Destinations favorites
   - Voyages favoris
3. Triez/filtrez si besoin
4. Chaque item a boutons:
   - Voir détails
   - Retirer du favori
   - Créer un voyage (si destination)
```
